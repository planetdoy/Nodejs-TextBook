http 서버가 있어야 웹 브라우저의 요청을 처리할 수 있으므로 http 모듈을 사용했습니다.

HTTP 상태코드
- 2XX
    - 00 성공
    - 01 작성됨
- 3XX
    - 01 영구 이동
    - 02 임시 이동
    - 04 수정되지 않음 (요청의 응답으로 캐시를 사용했다) 
- 4XX
    - 00 잘못된 요청
    - 01 권한 없음
    - 03 금지됨
    - 04 찾을 수 없음
- 5XX
    - 00 내부 서버 오류
    - 02 불량 게이트웨이
    - 03 서비스를 사용할 수 없음


요청 처리과정 중에서 에러가 발생했다고 해서 응답을 보내지 않으면 안됩니다.
성공/실패의 여부를 클라이언트 쪽에 요청이 마무리 되었음을 알려야 합니다.
그렇지 않으면 클라이언트는 한없이 기다리다 일정시간 후 Timeout(시간 초과) 처리합니다.
